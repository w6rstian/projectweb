CSS

Dowiesz się tutaj jak wykonać podstawowe czynności w CSS. Język CSS odwołuje się do znaczników HTML i pozwala nadać im określone atrybuty. CSS jest bardziej zaawansowany pod tym względem od HTML, dlatego stosuje się go do wykonywania szaty graficznej stron. Arkusze CSS zapisuje się w pliku z rozszerzeniem .css, wewnętrznie lub liniowo.

Dołączanie arkusza

Kod CSS można dołączyć na 3 sposoby. Im bliżej elementu definiowany jest atrybut, tym wyższy ma priorytet.

<head> <link rel="stylesheet" type="text/css" href="style.css"> <!-- Zewnętrzny arkusz stylów --> <style> /* Wewnętrzny arkusz stylów */ ... </style> </head> <body> <div style="..."> <!-- Liniowy arkusz stylów --> </div </body>

Składnia

CSS odnosi się do danego znacznika, identyfikatora lub klasy i nadaje mu atrybuty oraz ich wartości.

<style> div { background-color: tomato; } p { font-size: 12pt; color: white; } </style>

Paragraf

Kolory

Kolory w CSS są definiowane przez ich nazwy (red, orange, DodgerBlue), system RGB, system HEX lub system HSL. Wszystkie poniższe wartości wyznaczają ten sam kolor pomidorowy.

div { background-color: rgb(255, 99, 71); } h1 { color: #FF6347; } p { color: hsl(9, 100%, 64%); } section { border-color: tomato; }

Sekcja

Tła

W CSS każdy element może mieć swoje tło. Istnieje też skrótowy sposób zapisu, nie trzeba wtedy wymieniać każdego atrybutu po kolei.

body { background-color: #ffffff; background-image: url("img.png"); background-repeat: no-repeat; background-position: right top; background: #ffffff url("img_tree.png") no-repeat right top; }

Obramowania

Istnieją 2 typy obramowań. Border zajmuje miejsce i jest uwzględniany do szerokości obiektu. Outline jest rysowany na zewnątrz border i nie zmienia szerokości obiektu.

p { border-width: 1px; border-style: solid; border-color: black; } div { outline: 2px dashed #3300AA; } /* Jest możliwe też definiowanie każdej strony obramowania oddzielnie */ section { border-top: 1px dotted #FF6B6B; border-left: 2px solid #FFE66D; }

Marginesy

Margin definiuje margines zewnętrzny, a padding wewnętrzny.

div { margin: 0 auto; /* Pierwsza wartość ustala margines górny i dolny, druga lewy i prawy. */ } a { padding: 20px 0 10px; /* Górny, lewy i prawy, dolny */ margin: 10px 5px 0 auto; /* Górny, prawy, dolny, lewy */ color: lightblue; }

Tekst

Tekst można wyśrodkować za pomocą CSS.

p { color: white; /* Kolor czcionki */ text-align: center; /* Centrowanie */ text-decoration: underline; /* Podkreślenie */ } a { text-decoration: none; /* Usunięcie podkreślenia obecnego domyślnie przy linkach */ color: #FFE66D; }

Tryby wyświetlania

Element blokowy zawsze rozpoczyna się od nowej linii i zajmuje całą dostępną szerokość. Element inline nie rozpoczyna się od nowej linii i zajmuje tylko tyle miejsca ile potrzeba.

div:first-of-type { /* Pseudoklasa :first-of-type wybiera pierwszy element danego typu */ display: block; } div:nth-of-type(2) { display: none; /* Nie wyświetla się i nie zajmuje miejsca*/ } div:last-of-type { display: inline; }
